# Content & Writing Suite

All generated content is organized by type in these directories:

```
content/
├── blog/              # Blog posts (technical, business, thought leadership)
├── newsletters/       # Email newsletters (roundups, deep dives, announcements)
├── social/            # Social media (Twitter threads, LinkedIn, Instagram)
├── seo/               # SEO-optimized articles with keyword research
├── emails/            # Email campaigns (cold outreach, drip, launch, onboarding)
├── landing-pages/     # Landing page copy (hero, features, testimonials, CTA)
├── case-studies/      # Customer success stories (problem-solution-result)
├── strategy/          # Content strategy docs (calendars, pillars, personas)
├── repurposed/        # Content repurposed across platforms (1 → 10+)
└── edited/            # Professionally edited versions of content
```

## Anti-Cliche Policy

These phrases are **banned** from all generated content:
- "dive into", "deep dive", "let's explore"
- "unleash", "unlock", "supercharge"
- "game-changer", "paradigm shift", "disruptive"
- "in today's fast-paced world", "in the ever-evolving landscape"
- "it's worth noting", "it goes without saying"
- "leverage", "synergy", "ecosystem"
- "I'm excited to share", "Here's the thing"
- "Let that sink in", "Read that again"
